мүҪкІҢ л”°лқј л§Ңл“ңлҠ” Spring Batch (1)

Posted on 2019-05-05 Edited on 2020-12-15 In spring batch Disqus:

Spring BtachлһҖ

batchлҠ” мқјкҙ„мІҳлҰ¬мқҳ лң»мқ„ к°Җм§Җкі  мһҲлӢӨ. Spring BatchлҠ” AccentureмҷҖ Spring Sourceмқҳ кіөлҸҷ мһ‘м—…мңјлЎң 2007л…„м—җ нғ„мғқн–ҲлҠ”лҚ° Accentureмқҳ л°°м№ҳ л…ён•ҳмҡ° & кё°мҲ л Ҙкіј Spring н”„л Ҳмһ„мӣҢнҒ¬к°Җ н•©міҗм ё л§Ңл“Өм–ҙм§„ кІ°кіјл¬јмқҙлӢӨ.

к°„лӢЁнһҲ л§җн•ҙ, мҠӨн”„л§Ғ л°°м№ҳлһҖ мӢӨмӢңк°„мІҳлҰ¬к°Җ м•„лӢҲлқј н•ңлІҲм—җ мқјкҙ„мІҳлҰ¬ н•ңлӢӨлҠ” кІғмқ„ мқҳлҜён•ңлӢӨ. н•ңлІҲм—җ мІҳлҰ¬н•  лҢҖлҹүмқҳ лӮҙмҡ©(Job)мқ„ л“ұлЎқн•ҙлҶ“кі  нҠ№м • мӢңк°„м—җ к·ё jobмқ„ мқјкҙ„ мІҳлҰ¬н•ҳл©° мһ‘лҸҷн•ҳлҠ” л°©мӢқмқҙлқјкі  мғқк°Ғн•ҳл©ҙ нҺён•ҳлӢӨ.

мҳҲлҘј л“Өм–ҙ мӢӨмӢңк°„мңјлЎң мІҳлҰ¬н•ҳкё°м—җ л¶Җн•ҳк°Җ м»Өм„ң н•ҳлЈЁмқҳ лҚ°мқҙн„°лҘј лӘЁм•ҳлӢӨк°Җ мғҲлІҪм—җ мІҳлҰ¬н•ҳкұ°лӮҳ н•  л•Ң мӮ¬мҡ©н•ңлӢӨ.

мқҙлҹ° Spring BatchлҠ” Springмқҳ нҠ№м„ұмқ„ к°Җм§Җкі  мһҲм–ҙм„ң DI, AOP, м„ңл№„мҠӨ м¶”мғҒнҷ” л“ұ Spring н”„л Ҳмһ„мӣҢнҒ¬мқҳ 3лҢҖ мҡ”мҶҢлҘј лӘЁл‘җ мӮ¬мҡ©н•  мҲҳ мһҲмңјл©ҙм„ң, Accentureмқҳ Batch л…ён•ҳмҡ°к°Җ лӢҙкёҙ м•„нӮӨн…ҚмІҳлҘј мӮ¬мҡ©н•  мҲҳ мһҲлӢӨ.

Spring BatchлҠ” м—¬лҹ¬ JobмңјлЎң мқҙлЈЁм–ҙм ё мһҲлҠ”лҢҖ, м—¬кё°м„ң JobмқҖ н•ҳлӮҳмқҳ л°°м№ҳ мһ‘м—… лӢЁмң„лҘј лң»н•ңлӢӨ. Job м•Ҳм—җлҠ” м•„лһҳмІҳлҹј м—¬лҹ¬ Stepмқҙ мЎҙмһ¬н•ҳкі , Step м•Ҳм—җ Tasklet нҳ№мқҖ Reader & Processor & Writer л¬¶мқҢмқҙ мЎҙмһ¬н•ңлӢӨ.


Spring Btach л§Ңл“Өкё°

step 1. Spring boot н”„лЎңм қнҠё м„ нғқ

File-> New -> Project м—җм„ң Spring Initializr(Spring boot)лҘј м„ нғқн•ңлӢӨ.

step 2. Spring мқҳмЎҙм„ұ м„ нғқ

мҠӨн”„л§Ғ мқҳмЎҙм„ұмқ„ лӢӨмқҢкіј к°ҷмқҙ м„ нғқн•ңлӢӨ. лӢӨл§Ң мӮ¬мҡ©н•ҳлҠ” SQLм—җ л”°лқј м„ нғқн•ҳмһҗ.

step 3. н”„лЎңм қнҠё мғқм„ұ

м•һмқҳ л‘җлӢЁкі„лҘј кұ°м№ҳл©ҙ м•„лһҳмҷҖ к°ҷмқҙ DemoApplicationмқҙ мғқм„ұлҗҳкі  м•Ҳм—җлҠ” main л©”мҶҢл“ңк°Җ лӮҳнғҖлӮңлӢӨ.

step 4. м• л…ён…Ңмқҙм…ҳ м„Өм •

Spring BatchлҘј мӮ¬мҡ©н•ҳкё° мң„н•ҙм„  мҡ°м„  л°°м№ҳ кё°лҠҘмқ„ нҷңм„ұнҷ” н•ҳкё° мң„н•ҙ @EnableBatchProcessing мқ„ лӢ¬м•„мӨҳм•ј н•ңлӢӨ. н•ҙлӢ№ м• л…ён…Ңмқҙм…ҳмқҙ м—Ҷмқ„ кІҪмҡ° Spring Batchмқҳ кё°лҠҘмқ„ мӮ¬мҡ©н•  мҲҳ м—ҶмңјлӢҲ мЈјмқҳн•ҳмһҗ.

step 5. Job нҒҙлһҳмҠӨ л§Ңл“Өкё°

мқҙм ң н”„лЎңм қнҠёлҸ„ л§Ңл“Өм—ҲмңјлӢҲ Jobмқ„ мғқм„ұн•ҙліҙмһҗ. TestJob нҒҙлһҳмҠӨлҘј мғқм„ұн•ңлӢӨ.

step 6. Job л§Ңл“Өкё°

мғқм„ұ н•ҙлҶ“мқҖ TestJob нҒҙлһҳмҠӨм—җ мҪ”л“ңлҘј мұ„мӣҢліҙмһҗ.

  • @Slf4j
    лЎңк·ё мӮ¬мҡ©мқ„ мң„н•ң lombok м• л…ён…Ңмқҙм…ҳмқҙлӢӨ.
  • @RequiredArgsConstructor
    finalмқҙлӮҳ @NonNullмқё н•„л“ң к°’л§Ң нҢҢлқјлҜён„°лЎң л°ӣлҠ” мғқм„ұмһҗлҘј л§Ңл“Өм–ҙ мӨҖлӢӨ. мң„ нҒҙлһҳмҠӨм—җм„ңлҠ” JobBuilderFactory jobBuilderFactoryмҷҖ StepBuilderFactory stepBuilderFactoryлҘј л°ӣлҠ” мғқм„ұмһҗлҘј л§Ңл“ лӢӨ.
  • @Configuration(вҖңtestJobConfigurationвҖқ)
    Spring Batchмқҳ лӘЁл“  JobмқҖ @ConfigurationмңјлЎң л“ұлЎқн•ҙм„ң мӮ¬мҡ©н•ңлӢӨ.
  • jobBuilderFactory.get(вҖңtestJobвҖқ)
    testJob мқҙлһҖ мқҙлҰ„мқҳ Batch Jobмқ„ мғқм„ұн•ңлӢӨ.
    jobмқҳ мқҙлҰ„мқҖ лі„лҸ„лЎң м§Җм •н•ҳм§Җ м•Ҡкі , мқҙл ҮкІҢ BuilderлҘј нҶөн•ҙ м§Җм •н•ҳкІҢ лҗңлӢӨ.
  • stepBuilderFactory.get(вҖңtestStepвҖқ)
    testStepмқҙлһҖ мқҙлҰ„мқҳ Batch Stepмқ„ мғқм„ұн•ңлӢӨ.
    jobBuilderFactoryмҷҖ л§Ҳм°¬к°Җм§ҖлЎң BuilderлҘј нҶөн•ҙ мқҙлҰ„мқ„ м§Җм •н•ңлӢӨ.
  • .tasklet((contribution, chunkContext))
    Step м•Ҳм—җм„ң мҲҳн–үлҗ  кё°лҠҘл“Өмқ„ лӘ…мӢңн•ңлӢӨ.
    Stepм•Ҳм—җм„ң лӢЁмқјлЎң мҲҳн–үлҗ  кё°лҠҘл“Өмқ„ м„ м–ён• л•Ң мӮ¬мҡ©лҗҳлҠ”лҚ°
    мң„ мҪ”л“ңм—җм„ңлҠ” Batchк°Җ мҲҳн–үлҗҳл©ҙ log.info(вҖңThis is first StepвҖқ)мқҙ м¶ңл ҘлҗңлӢӨ.

step 6. Run

мқҙм ң Jobмқ„ л§Ңл“Өм—ҲмңјлӢҲ DemoApplicationмқ„ мӢӨн–үмӢңмјң ліҙмһҗ.
Runмқ„ нҒҙлҰӯн•ңлӢӨ.

step 7. мӢӨн–үкІ°кіј

мқҙм ң Consoleмқ„ нҷ•мқён•ҙліҙл©ҙ лӢӨмқҢкіј к°ҷмқҙ вҖҳThis is first StepвҖҷмқҙ м¶ңл ҘлҗҳлҠ” кІғмқ„ ліј мҲҳ мһҲлӢӨ.
